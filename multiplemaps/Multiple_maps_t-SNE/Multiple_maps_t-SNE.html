Multiple Maps t-SNE

 
 

The visualization tool on this page allows you to navigate maps that were produced using multiple maps t-SNE (see below). Such maps provide a way to visualize data of which the underlying similarities are non-metric, such as semantic similarities.


Multiple maps t-SNE is introduced in the following paper:

  1. • L.J.P. van der Maaten and G.E. Hinton. Visualizing Non-Metric Similarities in Multiple Maps. Machine Learning 87(1):33-55, 2012. [ PDF ]


Navigating in the tool works as follows:

  1. • Clicking on the bottom-left squares changes the map that is shown.

  2. • Clicking on a word in the visualization lights up the bottom-left squares to indicate in which map the selected word occurs.

  3. • Typing a word in the search box has the same effect.

  4. • Clicking on the Show all maps button shows all the maps that have the selected word in it.

  5. • Zooming and panning the maps works as expected.
